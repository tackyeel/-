# 主动发言与回复（中文QQ白名单版）

这是基于 `juzi865/initiative-talk-plugin` 改造的 MaiBot 主动私聊插件。

## 功能

- WebUI 配置字段使用中文标题。
- 支持 `QQ号白名单`，可以直接填写私聊对象 QQ 号。
- 保留 `会话hash白名单（高级）`，解析失败时仍可手动填 stream_id。
- 支持 `活跃时段开始/结束`，默认 `09:00-24:00`。
- 默认关闭“自动扫描所有会话”，避免误发到群聊或陌生私聊。

## 默认配置

- `QQ号白名单`: `[]`
- `检查间隔（分钟）`: `30`
- `主动发言概率`: `0.06`
- `活跃时段`: `09:00-24:00`
- `参考最近消息数`: `6`
- `失败重试次数`: `1`

## 安装

通过 MaiBot WebUI 从本仓库安装，或将本仓库内容放入 MaiBot 的 `plugins/` 目录后重启 MaiBot。

安装前建议先禁用其他主动发言插件，避免多个插件同时主动发消息。

## QQ号白名单说明

`QQ号白名单` 使用 SDK 的 `chat.get_stream_by_user_id(user_id, platform="qq")` 解析私聊会话。

如果某个 QQ 号解析失败，通常是因为对方还没有和麦麦产生过私聊会话。让对方先私聊麦麦一次，再重载插件即可。

## 配置建议

- 想省额度：把 `主动发言概率` 设置为 `0.03` 到 `0.08`。
- 想更活跃：可以把 `检查间隔（分钟）` 调到 `15`，但建议先观察额度消耗。
- 不建议开启 `允许自动扫描所有会话`，除非你明确希望它对所有已有群聊/私聊主动发言。
